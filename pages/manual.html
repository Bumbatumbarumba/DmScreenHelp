Select Campaign

Create New
Provides a user with a form to create a new campaign. Here, the user must specify a campaign name, and may specify the theme of their campaign; this field is optional. The date and time of creation is automatically gathered. No two existing campaigns can have the same name, although the theme can be repeated as many times as desired.

Load Selected
Once the user clicks a campaign from the list, this button is enabled and takes them to the DM Window screen.

Delete Selected
Once the user clicks a campaign from the list, this button is enabled and allows the user to delete the selected campaign. Before deleting, the user is prompted that this process is irreverable and are given a chance to cancel.

Map

The Map dropdown has two options: Create/Edit, and View.

Create/Edit
The program checks if there is an existing map, and loads that if it finds one; otherwise a form appears that allows the user to specify map dimensions. The map width and height must be an integer between 300 and 1500, inclusively, otherwise the user is prompted that their dimensions are incorrect.
Clicking Create on the form will produce a window containing several image editing tools, as well as a canvas of the specified dimensions. The following tools are available:

  • Draw Land - Changes the pen colour to be a sandy-beige colour, reminiscent of old maps, and changes the pen size to be 16px.
  • Draw Water - Changes the pen colour to be a light blue colour, and the pen size to be 16px.
  • Change Terrain Height - Raises the terrain height over land, and lowers the terrain height in water.
  • Free Style - Changes the pen colour to be black, and the pen size to be 1px.
  • Size - Changes the pen size to be the specified number of pixels; must be greater than zero.
  • Create Mountains - Changes the pen to be a simple mountain shape.
  • Create Towns - Special tool that creates radio buttons representing towns; has several parameter that can be specified.
  • Undo - Reverts the most recent stroke.
  • Redo - Returns the most recently reverted stroke.
  • Save - Saves the user's map progress.
  • Save and Close - Saves the user's map progress and closes the map editor.

View
If there is an existing map, then a window is generated displaying it.
